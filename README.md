# rmatic-relay

## start newEra task

``` bash
# without metric push
./build/rmatic-relay start --account xxx --eth_endpoint wss://goerli.infura.io/ws/v3/zzz --home workspace --stake_manager yyy
# with metric push
./build/rmatic-relay start --account xxx --eth_endpoint wss://goerli.infura.io/ws/v3/zzz --home workspace --stake_manager yyy --monitor_enable
```
